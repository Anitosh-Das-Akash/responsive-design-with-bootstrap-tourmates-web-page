🌿 Sylhet Tourism Website

A simple Sylhet Tourism Website built using HTML, CSS, and Bootstrap.
This project showcases some of the most popular places, restaurants, and monuments in Sylhet, Bangladesh.

The website includes a responsive navigation bar, image carousel, tourist information sections, and an accordion component.

🚀 Features

Responsive Navigation Bar

Image Carousel / Slider

Section for Restaurants

Section for Tourist Spots

Section for Monuments

Accordion component

Responsive layout using Bootstrap Grid

Clean and simple UI design

🛠️ Built With

HTML5

CSS3

Bootstrap 5
